# GitHub Setup Commands for Platform 5ive Repository

## Manual Git Setup Instructions

Since Replit has restrictions on automated Git operations, please run these commands manually in the Shell:

### 1. Remove any lock files (if they exist)
```bash
rm -f .git/index.lock .git/config.lock
```

### 2. Configure Git user
```bash
git config user.email "your-email@example.com"
git config user.name "Your Name"
```

### 3. Remove existing remote (if any) and add with token
```bash
git remote remove origin 2>/dev/null || true
git remote add origin https://$GITHUB_TOKEN@github.com/tsizzybots/platform_5ive_replit.git
```

### 4. Check current Git status
```bash
git status
```

### 5. Add all files to staging
```bash
git add .
```

### 6. Create commit with latest enhancements
```bash
git commit -m "Fixed QA email notification system and enhanced error handling

✅ Fixed email notification database errors - removed invalid customer_id field references
✅ Updated email templates to properly fetch customer data from Lead table
✅ Added comprehensive logging for email notification debugging
✅ Enhanced error handling with detailed API response logging
✅ Verified email delivery working - successful test with Resend API
✅ Fixed both HTML and text email template formatting

Technical Changes (August 13, 2025):
- Fixed MessengerSession model field references in email notifications
- Added Lead table lookup for customer information in QA emails
- Enhanced email error handling with detailed logging at each step
- Added API key validation and response debugging
- Tested and verified email delivery to team@izzyagents.ai
- Updated customer contact information display in email templates
- Added graceful error handling that doesn't break QA workflow

Previous Features:
- Enhanced conversation API with contextual filtering
- Real-time completion detection with multi-layer automation
- Robust session management and QA workflow
- Auto-sync system preventing orphaned sessions
- Dashboard with instant status updates
- PostgreSQL database with data integrity
- Embeddable chat widget for lead capture
- Webhook integration for automation workflows"
```

### 7. Push to GitHub
```bash
git push -u origin main
```

### If you get authentication errors, try these alternative approaches:

#### Option A: Use environment variable directly
```bash
git push https://$GITHUB_TOKEN@github.com/tsizzybots/platform_5ive_replit.git main
```

#### Option B: Update the remote URL with token
```bash
git remote set-url origin https://$GITHUB_TOKEN@github.com/tsizzybots/platform_5ive_replit.git
git push -u origin main
```

### Alternative: Create a new branch for safety
```bash
git checkout -b initial-commit
git push -u origin initial-commit
```
